Règles du Jeu de Mémoire

  1. Le jeu consiste à retourner des cartes, deux à la fois, pour trouver des paires identiques.
  2. Chaque carte est retournée et cachée après un court délai si elle ne forme pas une paire avec l'autre carte retournée.
  3. L'objectif est de retrouver toutes les paires en un minimum de coups.
  4. Le jeu se termine lorsque toutes les paires sont trouvées.
  5. Vous pouvez choisir de jouer avec un nombre différent de cartes selon votre niveau de difficulté.
